### 스타일링 가이드

#### 전역 스타일

전역 스타일 속성을 설정하려면, `settings.part.less` 파일을 열어 주석을 참고하여 수정하십시오. 전역 설정은 다음을 포함합니다:

* 전역 기본 폰트
* 기본 텍스트 색상
* Disabled 상태의 텍스트 색상
* 기본 보더 색상
* 포커스 상태의 보더 색상
* Disabled 상태의 보더 색상
* 내부에 아이템을 갖는 컨트롤의 경우, 각 아이템에 마우스 호버 시의 텍스트 색상
* 내부에 아이템을 갖는 컨트롤의 경우, 각 아이템에 마우스 호버 시의 배경 색상
* 내부에 아이템을 갖는 컨트롤의 경우, 각 아이템이 선택된 경우 텍스트 색상
* 내부에 아이템을 갖는 컨트롤의 경우, 각 아이템이 선택된 경우 배경 색상
* 내부에 아이템을 갖는 컨트롤이 Disabled된 경우, 각 아이템이 선택된 경우 텍스트 색상
* 내부에 아이템을 갖는 컨트롤이 Disabled된 경우, 각 아이템이 선택된 경우 배경 색상

#### 개별 컨트롤 스타일

개별 컨트롤의 스타일을 수정하려면, `theme/controls` 에 각각의 컨트롤에 해당하는 `.less` 파일을 열고 주석을 참고하여 수정하십시오.

### スタイリングガイド

#### グローバルスタイル

グローバルスタイル属性を設定するには、`settings.part.less` ファイルを開き、コメントを参考に修正してください。グローバル設定には以下が含まれます:

* グローバルデフォルトフォント
* デフォルトテキストカラー
* Disabled状態のテキストカラー
* デフォルトボーダーカラー
* フォーカス状態のボーダーカラー
* Disabled状態のボーダーカラー
* アイテムを含むコントロールの場合、各アイテムのマウスホバー時のテキストカラー
* アイテムを含むコントロールの場合、各アイテムのマウスホバー時の背景カラー
* アイテムを含むコントロールの場合、各アイテムが選択された時のテキストカラー
* アイテムを含むコントロールの場合、各アイテムが選択された時の背景カラー
* アイテムを含むコントロールがDisabled状態の場合、各アイテムが選択された時のテキストカラー
* アイテムを含むコントロールがDisabled状態の場合、各アイテムが選択された時の背景カラー

#### 個別コントロールスタイル

個別コントロールのスタイルを修正するには、`theme/controls` フォルダ内の各コントロールに対応する `.less` ファイルを開き、コメントを参考に修正してください。

### Styling Guide

#### Global Styles

To configure global style properties, open the `settings.part.less` file and refer to the comments for guidance. The global settings include:

* Global default font
* Default text color
* Text color in the disabled state
* Default border color
* Border color in the focused state
* Border color in the disabled state
* If a control contains items, the text color of each item when hovered
* If a control contains items, the background color of each item when hovered
* If a control contains items, the text color of each item when selected
* If a control contains items, the background color of each item when selected
* If a control that contains items is disabled, the text color of each item when selected
* If a control that contains items is disabled, the background color of each item when selected

#### Individual Control Styles

To modify the style of individual controls, open the corresponding `.less` file for each control in the `theme/controls` directory and follow the comments for instructions.
